## AI-Based News Aggregator

An **AI-powered News Aggregator** built using **Python and Streamlit** that collects, categorizes, and presents the latest news from various sources in a user-friendly interface. The application leverages **Natural Language Processing (NLP)** and intelligent algorithms to analyze news content, summarize articles, and provide personalized news recommendations.

### ✨ Features

* 📰 Real-time news fetching from multiple sources
* 🤖 AI-based news categorization and content analysis
* 📄 Automatic article summarization using NLP techniques
* 🔍 Keyword-based news search functionality
* 🧠 Smart recommendation of relevant articles
* 🌐 Clean and interactive Streamlit web interface
* 📊 Sentiment analysis of news articles
* 📱 Responsive and easy-to-use dashboard

### 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **AI/NLP:** NLTK, Transformers, TextBlob
* **Data Processing:** Pandas
* **APIs:** News API / Web Scraping
* **Machine Learning:** Scikit-learn

### 🚀 Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd AI-News-Aggregator

# Install dependencies
pip install -r requirements.txt

# Run the Streamlit application
streamlit run app.py
```

### 🎯 Project Goal

The goal of this project is to reduce information overload by delivering concise, relevant, and categorized news updates. By combining AI techniques with an intuitive Streamlit interface, users can quickly access meaningful news insights without browsing multiple websites.
